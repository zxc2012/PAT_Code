## Performance

Reciprocal Conception
- CPI: Clock cycles per Instruction
- MIPS: Millions of Instructions per Second

cpu time = Instructions * CPI * Clock cycle time

= Instructions * CPI / Clock rate

= Instructions / MIPS*10^6
