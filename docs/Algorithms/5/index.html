Backtracking

The basic idea is that suppose we have a partial solution (x1,...,xi)(x_1, ... , x_i )(x1​,...,xi​) where each xk∈Skx_k\in S_kxk​∈Sk​ for 1≤k≤i<n1\leq k\leq i<n1≤k≤i<n.

First we add xi+1∈Si+1x_{i+1} \in S_{i+1}xi+1​∈Si+1​ and check if (x1,...,xi,xi+1)(x_1, ... , x_i,x_{i+1})(x1​,...,xi​,xi+1​)satisfies the constrains. If the answer is “yes” we continue to add the next x, else we delete xi+1x_{i+1}xi+1​ and backtrack to the previous partial solution (x1,...,xi)(x_1, ... , x_i )(x1​,...,xi​).
