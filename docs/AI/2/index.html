Convolution Layer​

Convolutional Filters

padding: Preserve input spatial dimensions in output activations by padding with n pixels border

Input: W∗H∗DW * H * DW∗H∗D tensor

Hyperparameters:

  • Number of filters: K, K is usually a power of 2 (32, 64, ...)
  • Filter size: F∗F∗DF * F * DF∗F∗D, F is usually an odd number (e.g., 3, 5, 7)
  • Stride: S, S describes how we move the filter
  • Padding size: P

Output: W’∗H’∗KW’ * H’ * KW’∗H’∗K tensor

  • W’ = (W – F + 2P) / S + 1
  • H’ = (H – F + 2 P) / S + 1

One can stack convolution filters into a new tensor

Pooling Layer​

We can use Pooling as Subsampling

Input: W∗H∗DW * H * DW∗H∗D tensor

Hyperparameters:

  • Filter size: F∗FF * FF∗F
  • Stride: S

Output: W′∗H′∗DW' * H' * DW′∗H′∗D tensor

  • W' = (W – F) / S + 1
  • H’ = (H – F) / S + 1
